Posti Priority PortPaye tulostus.

== Lisenssi ==
Lisäosa on GPL lisensoitu. Lisäosaa saa ladata, muokata ja levittää täysin vapaasti. Lisätietoja lisäosan LICENSE tiedostossa.

== Asennus ==
1) Lataa .zip paketti
2) Kirjaudu WordPressin hallintaan
3) Valitse Lisäosat - Lisää uusi
4) Valitse .zip paketti koneeltasi, asenna lisäosa ja aktivoi se

== Käyttö ==
Tilaustenhallinnassa lisäosalla on mahdollista tulostaa PortPaye tarra valitulle tilaukselle.

== Tuki ==
Lisäosalle ei ole tarjolla tukea.

== Päivitykset ==
Lisäosaa ei kehitetä aktiivisesti, eikä näin ollen päivitetä säännöllisesti.
